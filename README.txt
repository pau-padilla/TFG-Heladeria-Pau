Proyecto: SGE Heladería - Gestión Integral Multisede
Autor: Pau Padilla Del Pozo

Contenido de la entrega:

- memoria_TFG_Pau.pdf
  Memoria final del proyecto.

- codigo_fuente/
  Carpeta con los archivos PHP de la aplicación web.

- base_datos/heladeria.sql
  Exportación de la base de datos utilizada en el proyecto.

Requisitos del sistema:

- Ubuntu Server
- Apache
- PHP
- MariaDB
- Navegador web

Ruta recomendada de despliegue:

Copiar los archivos PHP dentro de:

/var/www/html/heladeria

Base de datos:

Importar el archivo heladeria.sql en MariaDB para recrear la base de datos del proyecto.

Acceso a la aplicación:

Abrir en el navegador la IP del servidor seguida de la ruta del proyecto.

Ejemplo:

http://IP_DEL_SERVIDOR/heladeria/

Usuario de prueba:

admin

La aplicación permite iniciar sesión, consultar productos, añadir nuevos productos, editar productos existentes y eliminar registros desde la interfaz web.
